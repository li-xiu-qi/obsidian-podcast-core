# 快速开始使用播客生成器

## 简介

Obsidian Podcast Core 是一个 Obsidian 插件，利用大语言模型（LLM）和文本转语音（TTS）技术，将你的Markdown笔记转换为高质量的播客音频，帮助你轻松创建音频内容。

## 安装

1. 下载插件文件。
2. 将插件文件夹复制到你的 Obsidian vault 的 `.obsidian/plugins/` 目录下。
3. 在 Obsidian 中启用插件。

## 配置

1. 打开 Obsidian 设置。
2. 找到 "Podcast Core" 设置页面。
3. 输入你的 StepFun API 密钥（用于LLM和TTS）。
4. 选择你想要使用的模型（例如 step-1-flash）。
5. 可选：配置自定义 Base URL（用于代理或本地模型）。

## 使用

### 生成播客

- **工具栏按钮**：点击左侧工具栏的播客图标，打开播客生成面板。
- **命令面板**：使用命令 "Generate Podcast" 为当前文档生成播客。

### 步骤

1. 打开一篇笔记。
2. 点击生成播客按钮。
3. 选择生成模式：独白（Monologue）或对话（Dialogue）。
4. 配置音色（可选）。
5. 点击"生成播客"按钮。
6. 等待脚本和音频生成。
7. 播放生成的音频或导出为MP3。

### 示例

假设你有一篇笔记内容如下：

```markdown
# 机器学习简介

机器学习是人工智能的一个分支，它通过数据学习模式...
```

插件会分析内容并生成播客脚本，然后转换为音频。你可以选择独白模式，由单一讲述者叙述，或对话模式，由主持人和嘉宾对话。

## 注意事项

- 确保你的 API 密钥有效且有足够的使用额度。
- 插件支持中文和英文内容。
- 生成的音频会临时存储，可导出为MP3文件。
- 如果文档过长，插件会自动截取合适长度。
